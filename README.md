# Dashboard

A React + Vite dashboard starter project with Syncfusion UI components, Tailwind CSS, and modern tooling.

## Features

- React 19 application
- Vite build tooling for fast development
- Tailwind CSS integration
- Syncfusion component libraries included for charts, grids, forms, and scheduling
- ESLint setup for code quality

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

### Lint source files

```bash
npm run lint
```

## Project Structure

- `index.html` — HTML entry point
- `src/main.jsx` — React bootstrapping file
- `src/App.jsx` — Main application component
- `src/index.css` — Global styles and Tailwind imports
- `vite.config.js` — Vite configuration

## Notes

This repository currently includes the frontend setup only. Add backend or additional pages as needed.
